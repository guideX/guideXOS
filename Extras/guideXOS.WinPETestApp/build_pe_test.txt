Build from a VS 2022 x64 Native Tools prompt:

ml64 /c /Fo program.obj program.asm
link /NOLOGO /SUBSYSTEM:NATIVE /NODEFAULTLIB /ENTRY:entry /FIXED:NO /OUT:program.exe program.obj

Validate:
- dumpbin /headers program.exe -> Base Relocation Table present (not zero)
- dumpbin /imports program.exe -> no imports

Rename to .mue for guideXOS staging if not auto-copied:
- copy program.exe ..\Ramdisk\program.mue
